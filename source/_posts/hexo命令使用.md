## 需要安装的工具

- node.js
- git
- hexo安装:
```
$ npm install -g hexo-cli
```
- next:
[丰富的主题](https://hexo.io/themes/)


## hexo常用命令

#### 初始化init，新建一个网站。folder为空文件夹

```
$ hexo init [folder]
```

#### 生成静态文件

```
$ hexo generate 
or
$ hexo g 
```

#### 部署命令，用于将本地文件发布到git仓库

```
$ hexo deploy 
or
$ hexo d 
```

#### 启动本地服务，用于开发本地预览调试

```
$ hexo server 
or
$ hexo s 
```
#### 清除缓存文件和已存在静态文件

```
$ hexo clean 
```
